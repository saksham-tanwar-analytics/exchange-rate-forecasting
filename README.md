# Exchange Rate Forecasting & BI Reporting Platform

## Overview
Built an end-to-end forecasting and reporting pipeline to analyse and predict exchange rate movements for financial decision-making.

## Business Problem
Fluctuating exchange rates impact financial planning and require accurate forecasting and structured reporting.

## Solution
Developed a SQL-based data pipeline integrated with Power BI dashboards and time-series forecasting models.

## Impact
Delivered structured financial insights and forecasting outputs to support data-driven currency analysis.

## Key Features
- Time-series forecasting models
- SQL data pipeline for structured reporting
- Power BI dashboard for financial insights
- Trend analysis and forecasting visualisation

## Tech Stack
Python · SQL · Power BI · pandas · time-series models
